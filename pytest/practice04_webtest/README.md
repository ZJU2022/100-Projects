selenium支持web浏览器的自动化，主要由三个工具构成，WebDriver，IDE，Grid
WebDriver:使用浏览器提供的api来控制浏览器，像用户在操作浏览器
IDE:Chrome和firefox的插件，录制用户操作（基础工具,工作中用的不多）
Grid:分布式操作，可以在多个浏览器进行操作
